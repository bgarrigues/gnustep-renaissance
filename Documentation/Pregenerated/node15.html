2.1 The GNUstep Markup Language

The GNUstep Renaissance Markup Language (gsmarkup for short) is used to describe graphical user interfaces (windows, menus, etc). A gsmarkup file contains a description of how the graphical user interface should be created, which objects are to be placed in the user interface, and how these objects are interconnected between them and to other objects in the main application.
