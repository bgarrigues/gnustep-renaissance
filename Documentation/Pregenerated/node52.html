awakeFromGSMarkup informal protocol

This section contains a single method, which any object can implement to perform additional setup after it has been created from a GSMarkup file. 
- (void) awakeFromGSMarkup
This method is called in the top-level objects created from a GSMarkup file after all connections (outlets etc) have been established. You can implement this method to complete the setup of your objects.
